Minecraft

Two separate Minecraft-inspired creations: one in Python using Pyglet and the other in C using "modern" OpenGL shaders. I worked on these projects to learn more about 3-D programming.

Features

  • Uses modern GL techniques - no deprecated, fixed-function APIs are used.
  • Uses perlin / simplex noise to generate the terrain.
  • Uses sqlite3 to save terrain changes made by the user.
  • Experimental branch for shadow mapping.
  • Python version went "viral" and saw a lot of activity on GitHub.
